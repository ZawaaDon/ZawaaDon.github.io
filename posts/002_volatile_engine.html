Volatile Engine 技術資料ページ公開

2026-01-21 / Category: Game

📖 公開内容

現在開発中の自作ゲームエンジン「Volatile Engine」に関する技術資料ページを公開しました。 C++17とOpenGLを使用したゼロからのエンジン構築プロジェクトです。

→ Volatile Engine 技術資料ページ

🛠️ 技術スタック

  • 言語: C++17
  • グラフィックス: OpenGL 3.3 Core
  • 主要ライブラリ: SDL2, GLAD, GLM, stb_image, ASSIMP
  • ビルド: CMake + vcpkg

📊 開発状況

Phase 2: Engine Core が完了（2026/01/21）し、現在はPhase 3: Advanced Renderingに進行中です。

完成した主要システム:

  • ✅ Time Management System（フレーム非依存タイミング）
  • ✅ Input System（SDL2の抽象化レイヤー）
  • ✅ Scene Management（GameObject階層管理）
  • ✅ Asset Pipeline（ASSIMPによる3Dモデルロード）
